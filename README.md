IPL (2008-2025) Player Performance Dashboard

An interactive Power BI dashboard built to analyze the current IPL 2026 season. This project provides real-time insights into player and team performance through dynamic visualizations and key performance indicators (KPIs).

📌 Project Overview

The dashboard analyzes IPL 2026 data to uncover trends in batting, bowling, and team performance. Data was collected and processed using Python, enabling comprehensive season-wise and team-wise analysis.

🚀 Features
📊 Interactive and user-friendly Power BI dashboard
🏏 Analysis of player and team performance across all IPL franchises
📈 Tracks 15+ KPIs including:
Runs scored
Strike Rate
Batting Average
Wickets Taken
Economy Rate
Points Table Standings
🔍 Team-wise and player-wise performance comparison
📅 Current season insights and trends
🛠️ Tech Stack
Power BI
Python
Pandas
BeautifulSoup
Power Query
DAX
📂 Dataset

The dataset contains IPL 2026 season statistics, including player and team performance metrics collected through web scraping and data preprocessing.


📈 Key Insights
Identified top-performing batsmen and bowlers of the season.
Compared team performance across multiple KPIs.
Analyzed trends in batting and bowling performances throughout the tournament.
